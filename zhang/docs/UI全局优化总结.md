## 🎯 优化规划

### 阶段一：UI一致性优化（P0 - 立即实施）

#### 1. 动画系统统一
**问题**：动画时长不统一（0.2s, 0.3s, 0.4s, 0.5s, 0.6s, 0.8s）

**解决方案**：
- 使用 `lib/animation-system.ts` 中的标准时长
- 统一为：fast(150ms), normal(300ms), slow(500ms), dramatic(800ms)
- 统一缓动函数：standard, spring, fluid

**影响文件**：
- `components/wedding/hero-section.tsx`
- `components/wedding/gallery-section.tsx`
- `components/wedding/schedule-section.tsx`
- `components/wedding/interaction-section.tsx`
- `components/wedding/floating-ai-button.tsx`
- `components/wedding/page-navigation.tsx`

**预计时间**：2小时

#### 2. 设计令牌应用
**问题**：部分组件使用硬编码颜色、间距、阴影

**解决方案**：
- 统一使用设计令牌（design-tokens.ts）
- 替换所有硬编码值为Tailwind类名
- 建立使用规范文档

**预计时间**：3小时

#### 3. 响应式字体系统
**问题**：字体大小在小屏设备上体验不佳

**解决方案**：
- 建立响应式字体映射
- 应用响应式字体类（text-3xl md:text-5xl）
- 确保移动端最小字体16px

**预计时间**：2小时
