# 婚礼音乐文件夹

## 📁 目录说明

此目录用于存放婚礼音乐文件，按不同环节分类：

- **entrance/** - 进场曲目（5首）
- **tea/** - 敬茶曲目（5首）
- **dining/** - 用餐曲目（6首）
- **farewell/** - 送客曲目（4首）

## 🎵 如何添加音乐

请参考项目根目录的 `docs/音乐文件说明.md` 获取完整的音乐文件清单和使用说明。

### 快速开始

1. 准备MP3格式的音乐文件
2. 按照清单命名文件
3. 将文件放入对应的文件夹中
4. 刷新网站即可播放

## ⚠️ 注意事项

- 支持格式：MP3（推荐）、AAC、WAV、OGG
- 建议比特率：128-320 kbps
- 请确保有权使用这些音乐文件

## 💡 临时方案

如果暂时没有准备音乐文件，可以：

1. 使用播放器的"上传音乐"功能临时上传
2. 使用任意MP3文件进行测试
