## 5️⃣ 使用说明

### 5.1 媒体资源引用规范

1. **React组件中引用照片**：

   ```tsx
   import Image from 'next/image';
   
   // 使用示例
   <Image 
     src="/wedding/Bride-主纱1.jpg" 
     alt="新娘主纱照片" 
     width={800} 
     height={1200} 
     className="w-full h-auto object-cover" 
   />
   ```

2. **React组件中引用视频**：

   ```tsx
   // 使用示例
   <video 
     src="/MP4/wedding主婚1.mp4" 
     controls 
     className="w-full h-auto"
     poster="/wedding/wedding-主婚合影.jpg"
   >
     您的浏览器不支持视频标签
   </video>
   ```

3. **React组件中引用音乐**：

   ```tsx
   // 使用示例
   const audioRef = useRef<HTMLAudioElement>(null);
   
   <audio 
     ref={audioRef}
     src="/music/entrance/沫言 - 今天你要嫁给我.mp3" 
     controls 
     className="w-full"
   >
     您的浏览器不支持音频标签
   </audio>
   ```

### 5.2 性能优化建议

1. **图片懒加载**：使用Next.js的Image组件默认支持懒加载
2. **视频渐进式加载**：使用poster属性提供预览图
3. **音乐预加载**：根据环节预加载相关音乐
4. **响应式媒体**：为不同设备提供合适尺寸的媒体资源
