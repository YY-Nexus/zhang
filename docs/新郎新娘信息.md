## AI/自动化

**核心结论：为张波 & 邓芮婚礼构建一套“现代·智能·沉浸”婚礼网站，核心要素为：*3D 无框画廊、微透可拖拽弹窗（全功能）、中文音乐曲库、AI 智能交互与一键呼叫/导航*，并以强响应与科技感动画贯穿全局。**

### 设计概览

- **目标体验**：沉浸式首页（3D 无框画廊）→ 弹窗为主的功能入口（微透、可拖拽、移动、点击 X 或背景关闭）→ 全站统一 AI 智能交互按钮（左上 logo 底层映射），支持移动与桌面自适应。  
- **风格**：现代极简 + 温暖灯光质感，动效以流体过渡与微交互为主，表情与话语风格热情和谐，兼顾中西婚礼流程与洛阳习俗（六礼、迎亲、拜堂、敬茶等）.

---

### 弹窗模块实现（全功能）

- **通用特性**：微透背景；拖拽手柄；边界约束；可移动；支持键盘 Esc 与点击非弹窗区域关闭；动画：弹入缩放 + 背景模糊 + 阶段性阴影。  

- **模块化内容**：音乐播放器；RSVP 表单（餐饮偏好、座位）；留言墙与实时聊天室；来宾名册与签到；一键呼叫（拨号）与地图导航；媒体查看（3D 画廊内打开大图）。  

- **实现参考**：可借鉴现成可拖拽 Modal 实现与无障碍 ARIA 规范.

### 3D 无框画廊与交互

- **功能点**：Three.js 渲染画廊墙面；光照与反射；点击画作触发微透弹窗查看详情与播放对应音乐；第一/第三人称漫游与缩放预览；移动端降级为平面滑动视图.  
- **交互**：画廊与弹窗双向联动，投票选曲、点赞、实时留言同步显示。

### 中文音乐曲库与智能播放

- **首选曲库**：Mandopop 婚礼常用曲目合集（进场、用餐、敬茶、送客分段播放），支持来宾投票与情绪切换（浪漫/欢快/温馨）。  
- **智能规则**：根据环节自动切换播放列表；来宾投票优先级；淡入淡出与环绕音效。

### 技术栈与动画强化

- **前端**：Next.js + React + Three.js + GSAP；UI 组件可用 Tailwind/Antd 定制；弹窗拖拽用轻量库或自实现逻辑.  
- **后端**：Serverless API + WebSocket 实时通信；媒体 CDN；管理后台支持来宾导入与报表。  

**交付建议**：优先完成视觉原型与 3D 画廊核心交互（2 周），并行实现弹窗组件与音乐引擎（2 周），最后接入 RSVP 与实时功能（1–2 周）。

**精简结论：下面是一组可直接用于设计师、前端开发与AI生成器的中文提示词，覆盖**UI板块分区、浮动AI、微透可拖拽弹窗、响应式动效**与中文婚礼场景（洛阳习俗）**。  

### 提示词集合

- **首页布局**：设计一套现代极简首页，包含**3D无框画廊**、大幅倒计时、左上角Logo映射浮动AI按钮，风格：暖金+石墨灰，动效：流体过渡与光晕。  
- **板块分区**：创建分层式网格：Hero、关于我们、日程、画廊、互动、来宾管理；每层支持独立微交互与视差滚动。  
- **浮动AI**：左上Logo底层映射**智能交互按钮**，常驻微透图标，点击弹出可拖拽AI面板，支持语音、表情建议、流程引导与一键呼叫。  
- **弹窗模块**：实现**微透、可拖拽、移动、背景点击或Esc关闭**；动画：缩放+模糊+阴影分层；内含音乐、RSVP、留言、导航、来宾名单。  
- **音乐播放器**：中文婚礼曲库分段（进场/敬茶/用餐/送客），支持来宾投票、情绪切换与淡入淡出过渡。  
- **3D画廊交互**：Three.js墙面，点击画作弹窗详情并触发对应音乐与留言。  
- **响应式与无障碍**：移动优先降级策略、触控拖拽手势、ARIA标签与键盘导航。  
- **动画细节**：使用GSAP/FLIP实现流畅过渡；按钮反馈：微振动、色彩渐变、表情气泡。  
- **文案语气**：热情和谐、善用表情、兼顾中西礼序与洛阳习俗提示。
