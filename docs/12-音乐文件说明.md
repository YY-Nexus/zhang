## 🎼 音乐文件要求

### 格式要求

- **推荐格式**：MP3（兼容性最好）
- **备选格式**：AAC、WAV、OGG
- **不推荐**：FLAC、APE（文件太大，加载慢）

### 质量要求

- **比特率**：128-320 kbps（推荐 192 kbps）
- **采样率**：44.1 kHz
- **声道**：立体声（Stereo）

### 文件大小

- 单首歌曲建议：3-8MB
- 整个播放列表建议不超过：150MB

## 🆘 故障排除

### 音乐无法播放？

**1. 检查文件路径和命名**

```bash
# 查看entrance文件夹
ls -la public/music/entrance/

# 应该看到这5个文件：
# 沫言 - 今天你要嫁给我.mp3
# 沫言 - 给你们.mp3
# 沫言 - 只有你.mp3
# 沫语 - 爱你一万年.mp3
# 沫语 - 幸福的脸.mp3
```

**注意：** 文件名必须**完全一致**，包括：

- 艺术家名（沫言/沫语）
- 空格（ - 前后都有空格）
- 歌曲名（一字不差）

**2. 检查文件格式**

```bash
# 验证文件格式
file "public/music/entrance/沫言 - 今天你要嫁给我.mp3"
# 应该输出：Audio file with ID3 version 2...
```

**3. 检查浏览器控制台**

- 打开开发者工具（F12）
- 查看 Console 是否有错误
- 查看 Network 标签，确认音频文件是否成功加载（状态码 200）

**4. 清除缓存**

- 强制刷新：`Ctrl+Shift+R`（Windows）或 `Cmd+Shift+R`（Mac）
- 或清除浏览器缓存

### 音质不佳？

- 检查源文件质量（比特率是否过低）
- 使用更高比特率的 MP3（192kbps 或 320kbps）
- 避免多次转码（会损失音质）

### 文件太大，加载慢？

- 使用在线工具压缩 MP3（推荐：[mp3smaller.com](https://mp3smaller.com)）
- 降低比特率（不低于 128kbps）
- 剪辑掉不必要的前奏/尾奏
