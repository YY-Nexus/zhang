## 🔧 快速部署命令

### Vercel部署
```bash
# 安装Vercel CLI
npm i -g vercel

# 登录
vercel login

# 部署
vercel

# 生产环境部署
vercel --prod
```

### 服务器部署
```bash
# 1. 构建项目
pnpm build

# 2. 启动生产服务器
pnpm start

# 3. 使用PM2管理（推荐）
pm2 start npm --name "wedding-site" -- start
pm2 save
pm2 startup
```
